# 使用提示

# Office Tool Plus 快捷键

这些新的快捷键仅在 V9.0.2 等更新版本中可用。

  • Esc: 关闭右侧面板（如果已打开）
  • F1: 显示帮助
  • Ctrl + 1: 切换到主页
  • Ctrl + 2: 切换到部署页面
  • Ctrl + 3: 切换到激活页面
  • Ctrl + 4: 切换到工具箱页面
  • Ctrl + 5: 切换到文档转换页面
  • Ctrl + T: 显示设置页面
  • Ctrl + B: 显示关于页面
  • Ctrl + Shift + P: 打开命令框

# 部署页面

  • F5 & Ctrl + R: 刷新 Office 信息并重置配置
  • Ctrl + O: 打开配置文件
  • Ctrl + S: 保存配置文件到本地
  • Ctrl + I: 从网络导入配置文件
  • Ctrl + L: 安装 iSlide
  • Ctrl + Alt + P: 打开产品列表
  • Ctrl + Alt + L: 打开语言列表
  • Ctrl + Alt + D: 忽略警告或错误，强制进行部署

# 激活页面

  • F5 & Ctrl + R: 重新加载 Office 信息
  • Ctrl + D: 查看激活信息
  • Ctrl + E: 清除操作结果
  • Ctrl + Alt + P: 打开许可证列表

# 激活步骤

# 密钥激活（联网激活）

在密钥管理中输入你的 Office 产品密钥并点击“安装密钥”，然后点击 激活 即可。

如果安装密钥时显示“找不到产品 SKU”，先安装对应的许可证，然后再试。

# 电话激活（脱机激活）

在确保安装了正确的密钥后，在密钥管理的菜单中，点击 查看安装 ID，最后安装得到的确认 ID (CID) 即可。

如果安装密钥时显示“找不到产品 SKU”，先安装对应的许可证，然后再试。

# KMS 激活（需连接 KMS 主机）

在许可证管理中选择一个批量版 (Volume) 许可证并安装，接着在 KMS 管理中设置一个 KMS 主机，应用设置，然后点击 激活 即可。
